# Memory Scramble

A memory card matching game built with Python and Tkinter.

## Requirements
- Python 3.8+
- Tkinter (built into Python, no extra install needed)

## How to run
```
python main.py
```

## How to play
- Set the number of rows and columns (rows x cols must be even)
- Set how many seconds you want for the timer
- Click Start Game
- Click on two cards to flip them. If they match they stay up, if not they flip back
- Match all pairs before time runs out to win
