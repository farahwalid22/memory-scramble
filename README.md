# Memory Scramble

A memory card matching game built with Python and Tkinter.

## Requirements

- Python 3.8+
- Tkinter (built into Python, no extra install needed)

## How to run

```
python main.py
```

## How to play

- Set board size and timer
- Click two cards to flip them
- Match all pairs before time runs out
